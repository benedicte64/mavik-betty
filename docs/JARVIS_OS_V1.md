# Jarvis OS v1

## Mission
Jarvis est le moteur décisionnel de GCOS.

## Capacités v1
- Mémoire métier persistante
- Journal d'audit
- Gestionnaire de tâches
- Centre de notifications
- Base documentaire
- Analyse de procédures
- Suggestions décisionnelles avec validation humaine

## Modules connectés
- Atelier
- CRM
- Stock
- Planning
- Documents
- Finance

## Principes
- Ne jamais exécuter une action critique sans validation.
- Conserver la traçabilité des décisions.
- Fournir des explications pour chaque recommandation.
