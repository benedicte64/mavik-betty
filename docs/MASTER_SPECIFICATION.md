# MAVIK GCOS Master Specification

Ce document est la référence fonctionnelle et architecturale du projet.

## Principes
- Une donnée est saisie une seule fois.
- Aucun document n'est dupliqué.
- Tous les documents sont accessibles depuis les modules concernés.
- Maximum deux clics pour accéder à une information.
- Jarvis est intégré au cœur du système.

1. Architecture
2. Base de données
3. IA
4. UX/UI
