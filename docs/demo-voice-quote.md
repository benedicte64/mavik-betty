# Démonstration devis Jarvis

Scénario de test de référence :

> Crée un devis pour David Bourasseau, Ford Mustang GT V8 2020 bleu Velocity, 120 000 km, Pack Intégral Cryo + Dinitrol. Prépare le dossier, le devis visuel, le texte à valider, l’acompte de 50 % et les créneaux indicatifs.

Le devis reste bloqué avant envoi tant que David ou Bénédicte ne l’a pas validé.
