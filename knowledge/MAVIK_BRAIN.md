# MAVIK BRAIN

Version: 0.2

## Mission
MAVIK est un moteur de pilotage métier configurable.

- Profil Avenor / Betty : démonstration d'une société de création et de vente de logiciels.
- Profil GentleCarE / Jarvis : exploitation métier de GentleCarE avec ses données de production.

Les deux profils partagent le moteur, mais jamais leurs données.

## Principes
- Le code exécute.
- La base de connaissances explique.
- Les décisions sont historisées.
- Les procédures sont versionnées.
- Une action sensible reste soumise à validation humaine.
